# Random Cat Viewer

A simple and stylish web app that fetches a random cat from the FreeAPI public cats API and displays its details in a clean card layout. Users can fetch a new cat, open a detailed modal, and switch between dark and light themes.

## Features

- Fetches a random cat using the FreeAPI cats API
- Displays cat image, name, origin, and temperament
- Opens a modal with detailed cat information
- Includes dark and light theme toggle
- Responsive design for mobile and desktop screens
- Warm yellow-orange professional UI theme

## Tech Stack

- HTML
- CSS
- JavaScript
- FreeAPI public cats API

## How To Run

1. Download or clone this project.
2. Open the project folder.
3. Open `index.html` in your browser.
4. Click `Fetch New Cat` to load another random cat.

## API Used

```text
https://api.freeapi.app/api/v1/public/cats/cat/random
```

## Main Files

- `index.html` contains the page structure.
- `style.css` contains the full responsive UI styling.
- `script.js` handles API fetching, rendering, modal behavior, and theme switching.
